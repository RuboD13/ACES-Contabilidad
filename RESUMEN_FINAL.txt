================================================================================
SI ALGO FALLA
================================================================================

PROBLEMA: El build se queda pegado

SOLUCION:
- Es normal que PyInstaller tarde mucho sin mostrar progreso
- Si ha pasado más de 90 minutos, algo puede estar mal
- Reinicia el proceso: python build_exe.py

PROBLEMA: "No hay espacio suficiente"

SOLUCION:
- Necesitas 5-10 GB libres durante la compilación
- Limpia descargas, temp, papelera
- Verifica: disco C: con al menos 10 GB libres

PROBLEMA: Error de dependencias

SOLUCION:
- pip install --upgrade flask pandas pyinstaller
- Limpia: rm -rf build dist
- Reintenta: python build_exe.py
