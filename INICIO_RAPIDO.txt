================================================================================
SI ALGO FALLA
================================================================================

1. Verifica espacio en disco:
   - Necesitas al menos 5-10 GB libres para compilar
   - Espacio destino: ~300 MB

2. Actualiza las dependencias:
   pip install --upgrade flask pandas pyinstaller

3. Limpia y reintenta:
   - Borra las carpetas "build" y "dist"
   - Ejecuta nuevamente: python build_exe.py

4. Si PyInstaller falla:
   pip uninstall pyinstaller
   pip install pyinstaller
